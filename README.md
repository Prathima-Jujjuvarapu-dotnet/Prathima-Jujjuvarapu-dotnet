## 🏥 Pharmacy Management System

A **production-ready microservices architecture** for pharmacy operations, built with .NET 8 and Angular.

**Key features:**
- 🧩 8 microservices with Ocelot API Gateway  
- 🔐 JWT + OTP/QR dual authentication  
- 👥 6 admin roles with policy-based RBAC  
- 📊 Role-specific Angular dashboards  
- 📁 Shared DataAccessLayer with email & exception handling  

**Tech stack:**  
` .NET 8 ` ` Angular 17 ` ` C# ` ` TypeScript ` ` SQL Server ` ` Ocelot ` ` JWT `
